Porta padrão do MySQL: 3306

CRUD
C = Create = Criar = Inserir dados.
R = Read = Ler = Pegar dados.
U = Update = Atualizar = Modificar dados.
D = Delete = Excluir = Remover dados existentes.

Empresas que usam MySQL:
 - Facebook
 - Netflix
 - YouTube
 - Booking.com
 - Airbnb

Cliente
Cliente é um software para consultar o software original.
    Ex1: cliente = HeidiSQL (Windows) -> software = MySQL.
    Ex2: cliente = Sequel Pro (MAC) -> software = MySQL.

Banco de Dados
Banco de dados é como uma planilha de Excel, existem tabelas com dados que podem ser inseridos, obtidos, modificados, deletados, etc.
Uma planilha de Excel pode virar um banco de dados com uma importação em CSV, e vice-versa.

Colunas
Cada coluna tem informações sobre ela mesma:
 - nome da coluna
 - tipo da coluna
 - tamanho da coluna
 - se a coluna aceita NULL(nulo)
 - valor padrão

Tipos de dados mais comuns:
 - TINY / INT
 - FLOAT
 - VARCHAR / TEXT
 - DATE / TIME / DATETIME / TIMESTAMP (formatos: YYYY-MM-DD (DATE), HH:MM:SS (TIME))
