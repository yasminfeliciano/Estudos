Autofocus é o recurso que faz o cursor aparecer em um campo de input.
Size faz o campo de input ter um tamanho específico de caracteres, mas varia de acordo com o tamanho da letra, navegador, etc.
Maxlength faz o campo de input ter um certo número de caracteres fixo.
Required faz o campo ser requisitado antes de enviar o formulário.
AutoComplete faz o campo de input já aparecer preenchido com alguma informação que o Usuário já digitou antes. Normalize.css é uma biblioteca de css que padroniza as configurações, estilo, bugs e erros no navegador. Antigamente se usava o css reset que resetava todas as propriedades de todos os elementos, corrigia bugs e etc, mas agora a versão "moderna" é a Normalize.css.
