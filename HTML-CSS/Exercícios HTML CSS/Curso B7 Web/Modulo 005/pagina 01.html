fill distorce a imagem para caber nas medidas do campo


contain diminui ou aumenta a imagem para caber no campo sem distorcer, mesmo que sobre espaço


cover diminui ou aumenta a imagem para preencher todo o campo sem distorcer e cortando-a se necessário


none mantém o tamanho original da imagem e corta o que está para fora do campo


scale-down diminui a qualidade e o tamanho da imagem pra caber no campo sem distorcer, mesmo que sobre espaço
